Red Hat Storage delivers an innovative and open software solution for scale-out network attached storage (NAS) that helps enterprises cope with and manage the explosion of big, semi-structured, and unstructured data growth while capitalizing on the cost benefits of volume economics to reduce overall storage costs at scale. Whether deployed on-premise or in a private, public, or hybrid cloud environment, Red Hat Storage simplifies access to information and brings freedom of choice and the power of community-driven innovation to the enterprise.

Red Hat Storage Server

Red Hat Storage Server is a modern, unified, scale-out network attached storage (NAS) and object storage software solution that is designed to work seamlessly with industry-standard x86 servers. Built on the industry-leading Red Hat Enterprise Linux operating system, it provides freedom of choice to customers by allowing them to deploy cost-effective and highly available storage without compromising on scale or performance. Red Hat Storage Server can easily be deployed on-premise, in private clouds, in public cloud infrastructures, or in hybrid cloud environments and is optimized for storage-intensive enterprise workloads, including general-purpose file sharing, high-performance computing, nearline archival, and rich media content delivery.

Deployment Options

  • Red Hat Storage Server for On-premise

    Scalable, high-performance storage for your data center or on-premises private cloud.

  • Red Hat Storage Server for Public Cloud

    Fully POSIX compatible, highly available, storage solution for public cloud storage using the Amazon Web Services (AWS) public cloud infrastructure.

  • Red Hat Storage Server for Hybrid Cloud

    Scalable, highly available storage solution unifying your on-premise data center/private cloud with public cloud infrastructures. Ideal for disaster-recovery purposes.
